Projects

Traffic Signal Detection using Yolo Algorithm

Developed a real-time traffic signal detection system using the YOLO (You Only Look Once) algorithm to enhance road safety and optimize traffic flow. The system accurately detects traffic lights from live video feeds and helps drivers make informed decisions. This project secured second prize at the "Science Expo 2K22" due to its innovative approach in reducing congestion and promoting efficient urban mobility.

Deep Learning based Behaviour Monitoring in Classroom

Designed a deep learning model to monitor student behavior in classrooms, using computer vision techniques to track attention levels and body posture. The project aimed to assist teachers in understanding student engagement and identifying those who might need additional support. This system provides insights that could improve teaching strategies and enhance learning outcomes.

Analysis of Deep Learning based Optimization Techniques for Oral Cancer Detection

Conducted an in-depth analysis of various deep learning optimization techniques to enhance the accuracy of oral cancer detection using medical imaging. The study focused on comparing different algorithms to find the most effective model for early diagnosis, thereby aiding in timely treatment. This research contributes to the medical field by potentially increasing survival rates through improved diagnostic methods.

Loan Amount Prediction Using Multivariate Linear Regression

Developed a predictive model for determining loan amounts using multivariate linear regression. The model leverages borrower income, credit scores, and demographic data to improve loan approval decision-making. This project provides valuable insights into key factors affecting loan approvals, helping financial institutions streamline their decision-making processes.
